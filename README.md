## 功能介绍

- 完美支持战地全系列和其他部分EA游戏完全脱离EA Desktop/Origin启动游戏
- 完全使用原生方式启动游戏，不再需要修改游戏文件
- 完美兼容寒霜mod的使用
- 完美兼容带有eaac反作弊游戏
- 支持多开游戏
- 多账号功能，支持100个账号切换
- 高级日志功能
- 自定义游戏启动项，支持注册表和手动选择
- 切换游戏语言
- 可不经过steam直接启动ea游戏
- 可绕过ea账号对于语言的限制

## 已知问题
